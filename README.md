
# Hackathon Booking

A booking application for managing hackathon registrations and event scheduling.

## Features

- User registration and authentication
- Event booking and management
- Schedule management
- Participant tracking

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd hackathon-booking
npm install
```

### Usage

```bash
npm start
```
